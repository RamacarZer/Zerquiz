# Zerquiz Frontend

React 18 + TypeScript + TailwindCSS ile geliştirilmiş Zerquiz platformunun kullanıcı arayüzü.

## Kurulum

```bash
cd frontend/zerquiz-web
npm install
```

## Geliştirme

```bash
npm run dev
```

Tarayıcıda http://localhost:3000 adresinden erişilebilir.

## Build

```bash
npm run build
```

## Özellikler

- React 18
- TypeScript
- TailwindCSS
- React Router
- React Query
- Zustand (state management)
- Axios (API client)

## Sayfa Yapısı

- `/` - Login
- `/dashboard` - Dashboard
- `/questions` - Soru Bankası
- `/exams` - Sınavlar
